# Task Management System

This project was developed in **Java** to practice software design, data handling, and object-oriented programming concepts.  
It is a **console-based task management system** that allows users to create, list, update, and delete tasks efficiently.

---

## About the Project

The goal of this project is to simulate a simple yet functional **task management application**, similar to a to-do list.  
The user can manage daily tasks directly through the terminal â€” adding new ones, marking them as completed, or removing them when done.

This project demonstrates how Java can be used to build practical and structured applications using core programming principles.

---

## Features

- Add a new task  
- View all tasks  
- Mark a task as completed  
- Delete a task  
- Display pending and completed tasks separately  

---

## Technologies Used

- **Java**
- **Scanner** (for handling user input)
- **ArrayList** (for managing the list of tasks)
- **Object-Oriented Programming (OOP)** principles
